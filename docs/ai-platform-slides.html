怎麼做到 · AI 平台

Azure IoT Edge＝把模型送上邊緣的控制塔

雲端負責打包、版本化、部署模型；推論仍在邊緣執行。

雲端 · Azure IoT Edge
控制塔 — 管理模型生命週期
打包 — posture-api 容器＝一個 IoT Edge 模組
版本化 — Azure Container Registry 存放模組映像
部署 — IoT Hub 下發部署清單到邊緣節點
更新 — 回流 → 重訓 → 一鍵把新模型重部署
邊緣節點
執行推論 — 模型真正跑的地方
即時推論 — 撞擊當下毫秒級判讀，不必等雲端
可離線 — 斷網也能偵測，不靠雲端連線
資料在地 — 感測資料留在邊緣，隱私安全
推論刻意放邊緣；雲端平台只負責把最新模型打包、送上來、保持同步——這就是它作為「AI 平台」的價值。
